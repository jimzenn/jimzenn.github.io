{{% definition name="Joint PMFs" status="" %}}

Let $\b{X} = (X\_1, X\_2, ..., X\_n)$ be a vector of $n$ discrete random variables,. Let $\b{x}  = (x\_1, x\_2, ..., x\_n) $ be possible values of the random variables, the PMF of the event

$$\set{ \b{X} = \b{x}} = \set{X\_1 = x\_1, X\_2 = x\_2, ..., X\_n = x\_n}$$

or more precisely

$$\bigcap\_{i=1}^n \set{X\_i = x\_i}$$

is the **joint PMF**, denoted as follow

$$p\_{ \b{X}}(\b{x}) = \b{P}(\b{X} = \b{x})$$

The joint PMF determines the probability of any event $A$ that can be specified in terms of the random variables $X\_1, X\_2, ..., X\_n$

$$\b{P}(\b{X} \in A) = \sum\_{ \b{X} \in A } p\_{ \b{X}}(\b{x})$$

{{% /definition %}}

{{% definition name="Marginal PMFs" status="" %}}

Given a joint PMF $p\_ \b{X}$, we call $p\_{X\_i}(x\_i), i \in \set{ 1, 2, ..., n }$ the **marginal PMF**

$$p\_{X\_i}(x\_i) = \sum\_{X\_1} ... \sum\_{X\_{i-1}}\sum\_{X\_{i+1}}... \sum\_{X\_n} p\_{\b{X}} (\b{x})$$

{{% /definition %}}

{{% definition name="Functions of Multiple Random Variables" status="" %}}

Let $X\_1, X\_2, ..., X\_n$ be $n$ discrete random variables denoted as $\b{X}$. $Y$ be another random variable defined as $Y = g(\b{X})$. Its PMF can be calculated from the joint PMF according to

$$p\_Y(y) = \sum\_{\set{ \b{x} | g(\b{x}) = y}} p\_{ \b{X}}(\b{x})$$

The expected value rule for functions naturally extends and takes the forms

$$E [g(\b{X})] = \sum\_{X\_1} \sum\_{X\_2}... \sum\_{X\_n} g(\b{x}) p\_{\b{X}}(\b{x})$$

{{% /definition %}}
