{{% definition name="Joint PDFs" status="" %}}

Two continuous random variables associated with the same experiment are **jointly continuous** and can be described in terms of a **joint PDF** $f\_{X,Y}$ if $f\_{X,Y}$ is a nonnegative function that satisfies

$$P((X, Y) \in B) = \iint\_{(x, y) \in B } f\_{X,Y}(x, y) \d x \d y$$

for every subset $B $ of the two-dimensional plane.

$\int\_{ -\infty }^{ +\infty } \int\_{ -\infty }^{ +\infty } f\_{X, Y}(x, y) \d x \d y = 1$(Normalization)

{{% /definition %}}

{{% definition name="marginal PDF" status="" %}}

The **marginal PDF** $f\_X$ of $X $ is given by

$f\_X(x) = \int\_{ -\infty }^{ +\infty } f\_X,Y(x, y) \d y$

Similarly,

$f\_Y(y) = \int\_{ -\infty }^{ +\infty } f\_X,Y(x, y) \d x$


{{% /definition %}}

{{% definition name="Joint Expectation" index="" status="" %}}

If $g: \R \times \R \to \R $, the expectation is:

$$E[g(X, Y)] = \int\_{ -\infty}^{ +\infty } \int\_{ -\infty }^{ +\infty } g(x, y) f\_{X, Y}(x, y) \d x \d y $$

For any scalar $a$, $b$, and $c$, we have

$$E[aX + bY + c] = aE[X] + bE[Y] + c$$

{{% /definition %}}


{{% definition name="Joint CDFs" status="" %}}

If $X $ and $Y $ are two random variables associated with the same experiment, we define their joint CDF by

$F\_{X, Y}(x, y) = P(X \leq x, Y \leq y) = \int\_{ -\infty }^{ x } \int\_{ -\infty }^{ y } f\_{X, Y}(s, t) \d t \d s$

{{% /definition %}}

{{% definition name="independence" status="" %}}

If $X, Y $ are jointly continuous random variables, they are independent if:

$$f\_{X,Y}(x, y) = f\_X(x)f\_Y(y) $$

This implies,

for any subsets $A \subseteq \R, B \subseteq \R,$

$$P(x \in A, y \in B) = P(x \in A) P(y \in B)$$

Also:

$$E[XY] = E[X]E[Y] $$

{{% /definition %}}
