{{% definition name="independent pair of events" %}}
Let $A,B \subset \Omega$, we say $A$ and $B$ are **independent** if
$$\b{P}(A\cap B)=\b{P}(A)\b{P}(B)$$

It follows that if $A, B$ are independent, then $\b{P}(A|B)=\b{P}(A)$.
{{% /definition %}}

{{% definition name="independence of multiple events" %}}

We say $A\_1, ..., A\_n$ are **pairwise independent** if

$$\forall i, j \in \set{ 1, 2, ..., n }, \b{P}(A\_i A\_j) = \b{P}(A\_i)\b{P}(A\_j) $$

We say $A\_1, ..., A\_n$ are **independent** if

$$\forall S \subseteq \set{1,2,...,n}, \b{P}(\bigcap\_{i\in S} A\_i) = \prod\_{i\in S}{A\_i}$$

{{% /definition %}}

{{% definition name="Conditionally independent" %}}

Let $C$ be an event with $P\(C) \neq 0$, $A, B \subset \Omega$ are **conditionally independent with respect to C** means

$$\b{P}(A\cap B | C) = \b{P}(A|C)\b{P}(B|C)$$

{{% /definition %}}
