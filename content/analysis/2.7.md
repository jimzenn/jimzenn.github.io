{{% definition name="sequence" status="" %}}

A **sequence** is <u>function</u> whose domain is a set of the form $\set{ n \in \Z \mid n \geq m } $; $m =0$ or $m = 1$. We denote the sequence by a letter such as $s $ and denote its value at $n $ as $s\_n $.

{{% /definition %}}

{{% definition name="limit" status="" %}}

A sequence $(s\_n) $ of real numbers is said to **converge** to the real number $s $ provided that

for each $\epsilon > 0$ there exists a number $N $ such that

$$n > N \to \abs{ s\_n -s } < \epsilon $$

If $(s\_n) $ converges to $s$, we will write $\limu{ n }{ \infty }s\_n = s $, or $s\_n \to s $.

The number $s $ is called the **limit** of the sequence $(s\_n) $.

A sequence that does not converge to some real number is said to **diverge**.

{{% /definition %}}

{{% example name="" %}}

$s\_n = \frac{ 1 }{ n }, \limu{ n }{ \infty } s\_n = 0 $

{{% proof index="" name="" %}}

Want to find some $N$, s.t. $n \geq N, \abs{ \frac{ 1 }{ n } - 0} < \epsilon, $

$$\frac{ 1 }{ n } < \epsilon \iff n > \frac{ 1 }{ \epsilon } $$

want $N = \frac{ 1 }{ \epsilon }$

Let $N = \ceil{ \frac{ 1 }{ \epsilon }} $

N be some integer > $\frac{ 1 }{ \epsilon }$

For $n > N $, $\frac{ 1 }{ n } - 0 = \frac{ 1 }{ n } $

$\frac{ 1 }{ N } < \frac{ 1 }{ \frac{ 1 }{ \epsilon }}  = \epsilon $.

Thus $\limu{ n }{ \infty } 0 $.

{{% /proof %}}

{{% /example %}}
