{{% definition name="Pointwise convergence" status="" %}}

Let $(f^{(n)})^\infty\_{ n =1} $ be a sequence of functions from one metric space
$(X, d\_X) $ to another $(Y, d\_Y) $, and let $f: X \to Y $ be another function. We say that $(f^{(n)})^\infty\_{ n =1} $ **converges pointwise** to $f $ on $X$ if we have

$$\limu{ n }{ \infty } f^{(n)} (x) = f(x) $$

for all $x \in X $, i.e.

$$\limu{ n }{ \infty } d\_Y(f^{(n)}(x), f(x)) = 0 $$

{{% /definition %}}

{{% definition name="" status="" %}}

$f\_n \to f$ uniformly, iff

$\forall \epsilon > 0, \exists N \in \N, \forall n \geq N, \forall x \in X $

$$d\_Y(f\_n(x), f(x)) < \epsilon $$

{{% /definition %}}

{{% example name="" %}}

Let $X=[0,1] $ and $Y = \R $($d\_{l^2}$)

Then $f\_n(x) = x^n $ converges to

$$f(x) =
\begin{cases}
0 &\text{ if } 0 \leq x < 1 \br
1 &\text{ if } x = 1
\end{cases}$$

{{% proof index="" method="" %}}

(Pointwise) If $0 \leq X < 1$ then it is clear.

$\limu{ n }{ \infty } f\_n(x) = \limu{ n }{ \infty }x^n = 0 $

but if $x = 1$, the limit is $1$.


(Uniform) WTS that the uniform convergence fails. Here's one way

If I fix some $\epsilon > 0 $, I want to show that for any choice of
$N \in \N $.

I have some $x \in [0, 1] $ such that $\abs{ f\_N(x) - f(x)} \geq \epsilon $

fix some $0 < \epsilon , 1 $, and take $N \in \N $ any integer, choose

$x = \epsilon^{ \frac{ 1 }{ n }} \in (0, 1)$

Then note


$$\begin{align}
\abs{ f\_n(x) - f(x)}
&= \abs{(\epsilon^{ \frac{ 1 }{ n }})^n - 0 }
&= \abs{ \epsilon } \geq \epsilon
\end{align}$$

$\implies f\_n \not \to f $ uniformly

{{% /proof %}}

{{% proof index="" method="" %}}

Since $f\_n $ some continuous (and f is not continuous). Then $f\_n \to f $ uniformly is impossible.

{{% /proof %}}
