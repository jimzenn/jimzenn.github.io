{{% definition name="Trigonometric functions" status="" %}}

$z \in \C.$

$$\cos z \triangleq \frac{ e^{iz} + e^{-iz}}{ 2 } $$

$$\sin z \triangleq \frac{ e^{iz} - e^{-iz}}{ 2 } $$

{{% /definition %}}

{{% remarks name="" %}}

$$e^{iz} = 1 + iz - \frac{ z^2 }{ 2! } - \frac{ iz^3 }{ 3! } + \frac{ z^4 }{ 4! } + ... $$
$$e^{-iz} = 1 - iz - \frac{ z^2 }{ 2! } + \frac{ iz^3 }{ 3! } + \frac{ z^4 }{ 4! } - ... $$
$$\cos z = 1 - \frac{ z^2 }{ 2! } + \frac{ z^4 }{ 4! } - ... = \sum\_{ n=0 }^{ \infty } \frac{(-1)^n z^{2n}}{(2n)! } $$
$$\sin z = 1 - \frac{ z^3 }{ 3! } + \frac{ z^5 }{ 5! } - ... = \sum\_{ n=0 }^{ \infty } \frac{(-1)^n z^{2n+1}}{(2n+1)! } $$

{{% /remarks %}}

{{% theorem name="Trigonometric identities" index="" status="" %}}

Let $x, y$ be real numbers.

1. $\sin ^2 x + \cos ^2 x = 1 $ <br>
$\sin x \in [-1, 1], \cos x \in [-1, 1], \forall x \in \R $
2. $\sin' x = \cos ' x. $ <br>
$\cos' x = - \sin x$.
3. $\sin (-x) = - \sin x$ and $\cos (-x) = \cos x $
4. $\cos (x + y) = \cos x \cos y - \sin x \sin y$ <br>
$\sin (x + y) = \sin x \cos y + \cos x \sin  y $
5. $\sin 0 = 0, \cos 0 = 1$
6. $e^{ix} = \cos x + i \sin x $ <br>
$e^{-ix} = \cos x - i \sin x $ <br>
$\cos x = \Re{ e^{ix}}$  <br>
$\sin x = \Im{ e^{ix}}$

{{% /theorem %}}

{{% lemma name="" index="" %}}

$\exists x > 0, \sin x = 0$.

{{% /lemma %}}

{{% definition name="\pi" status="" %}}

$\pi \triangleq \infi{ \set{ x \in (0, \infty): \sin x = 0 }} $

{{% /definition %}}

{{% theorem name="Periodicity of Trigonometric functions" index="" status="" %}}

$x \in \R $

1. $\cos (x + \pi) = - \cos x $ <br>
$\sin (x + \pi) = - \sin x $ <br>
$\sin x = 0 \iff \frac{ x }{ \pi } \in \Z$ <br>
$\cos x = 0 \iff \frac{ x }{ \pi } - \frac{ 1 }{ 2 } \in \Z $


{{% /theorem %}}
