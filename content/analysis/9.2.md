{{% definition name="Direct Sum" status="" %}}

$f: X \to Y, g: X \to Z$. We define their **direct sum** 

$$f \oplus g : X \to Y \times Z $$

as 

$$f \oplus g (x) := (f(x), g(x))$$

{{% /definition %}}

{{% lemma name="direct sum operation preserves continuity" index="" %}}

$f: X \to \R, g: X \to \R.$ We give $\R^2$ the Euclidean metric.

1. If $x\_0 \in X$, then $f$ and $g$ are both continuous at $x\_0$ if and only if $f \oplus g$ is continuous at $x\_0$.

2. $f$ and $g$ are both continuous if and only if $f \oplus g$ is continuous.

{{% /lemma %}}

{{% lemma name="" index="" %}}

1. The addition function $(x, y) \mapsto x + y$

2. the subtraction function $(x, y) \mapsto x - y$

3. the multiplication function $(x, y) \mapsto xy$

4. the maximum function $(x, y) \mapsto \max{ x, y }$

5. the minimum function $(x, y) \mapsto \min{ x, y }$

are all continuous functions from $\R^2$ to $\R$.

The division function $(x, y) \mapsto x / y $ is a continuous function from $\R \times \R \setminus \set{ 0 }$ to $\R$.

$\forall c \in R$, the function $x \mapsto cx$ is a continuous function from $\R$ to $\R$.

{{% /lemma %}}

{{% corollary name="" index="" %}}

Let $(X, d) $ be a metric space, let $f: X \to \R $ and $g: X \to \R $ be functions. Let $c $ be a real number.

1. If $x\_0 \in X $ and $f $ and $g $ are continuous at $x\_0 $, then the functions
$f+g: X \to \R, f-g: X \to \R, fg: X \to \R, \max{ f,g }: X \to \R, \min{ f,g }: X \to \R, $ and $cf : X \to \R $ are also continuous at $x\_0$. If $\forall x \in X, g(x) \neq 0$, then $f/g: X \to \R $ is also continuous at $x\_0 $.

2. If $f$ and $g$ are continuous, then the functions $f + g : X \to \R $, $f -g : X \to \R $, $fg: X \to \R$, $\max{ f,g }: X \to \R $, $\min{ f,g }: X \to \R $, and $cf : X \to \R $ are also continuous at $x\_0$. If $\forall x \in X, g(x) \neq 0$, then $f/g : X \to \R $ is also continuous at $x\_0$.

{{% /corollary %}}
