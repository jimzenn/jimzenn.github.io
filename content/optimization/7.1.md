{{% definition name="unimodal" status="" %}}

If $f $ has only one local minimizer, we say the function $f $ is unimodal.

{{% /definition %}}

{{% definition name="Golden section search" status="" %}}

**Golden section search** is a search method that can narrow down the minimizer $x^\*$ of a <u>unimodal</u> function $f: \R \to \R $ over a <u>closed</u> interval $[a\_0, b\_0]$.

Golden section search is an <u>iterative</u> method, and is defined as follows:

Let $\rho = \frac{ 3 - \sqrt[  ]{ 5 }}{ 2 } = 1 - .6180339887... $

1. Let $l\_n = b\_n - a\_n, s^\pare{n}\_a = l\_n (1- \rho), s^\pare{n}\_b = l\_n \rho$.

2. Compare $f(s^\pare{n}\_a)$ and $f(s^\pare{n}\_b) $:

  1. if $f(s^\pare{n}\_a) \geq f(s^\pare{n}\_b)$ then $x^\* \in [a\_n, s^\pare{n}\_b]$. <br> set $a\_{n+1} = a\_n, b\_{n+1} = f(s^\pare{n}\_b) $. Iterate. Notice that $s^\pare{n+1}\_a = s^\pare{n}\_a$.

  2. if $f(s^\pare{n}\_a) < f(s^\pare{n}\_b)$ then $x^\* \in [s^\pare{n}\_a, b\_n]$. <br> set $a\_{n+1} = f(s^\pare{n}\_a), b\_{n+1} = b\_n $, iterate. Notice that $s^\pare{n+1}\_b = s^\pare{n}\_b$.


$N$ steps of reduction using the golden section method reduces the range by the factor

$$(1 - \rho)^N \approx (0.61803)^N $$
