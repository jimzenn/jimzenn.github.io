Is there a more quantitative / analytical way of determining stability at a fixed point?

Let $x^{\*} $ be a fixed point. Define a small perturbation.

$\eta(t) = x(t) - x^{\*} $

To see if the perturbation grows or decays, take the derivative with respect to (w.r.t.) $\dot{ x }  = f(x) $

$\frac{ \d \eta }{ \d t } = \frac{ \d }{ \d t } [ x(t) - x^{ \* } ] = \dot{ x } = f(x) = f(x^{\*} + \eta)$

Using Taylor expansion about $x^\* $, we know that

$f(x^\* + \eta) = f(x^\*) + \eta f^\prime(x^\*) + O(\eta^2)$

$f(x^\*) = 0$ since x^\* is a fixed point.

we can neglect these terms if $f\prime(x^\*) \neq 0 $

$f(x^\* + \eta) = \dot{ \eta } \approx f^\prime(x^\*)$

- grow when $f'(x^\*) > 0 $
- decrease when $f'(x^\*) < 0 $
